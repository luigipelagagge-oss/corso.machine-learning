Versione Italiana (Abstract)

Questo sito è allo stesso tempo un tutorial e il proprio caso di studio: mentre guida il lettore attraverso i fondamenti dell’HTML e della struttura di una moderna pagina web, mostra in pratica come esso stesso è stato realizzato. Accanto all’introduzione allo sviluppo web, il sito propone un primo incontro con il machine learning, presentato attraverso un semplice esercizio di laboratorio di fisica. A partire dal calcolo della corrente in una resistenza, e successivamente estendendo il modello con l’introduzione di un diodo, l’utente scopre come un sistema fisico elementare possa diventare la base per l’analisi dei dati e la modellazione predittiva. Unendo programmazione pratica ed esperimenti scientifici, il sito invita i lettori curiosi a esplorare come strumenti digitali e algoritmi intelligenti possano nascere dai principi più semplici.
English Version (Abstract)

This website is both a tutorial and its own case study: while guiding the reader through the essentials of HTML and the structure of a modern web page, it simultaneously demonstrates how it was built. Alongside this introduction to web development, the site offers a first encounter with machine learning, presented through an accessible physics laboratory exercise. Starting from the computation of the electric current through a resistor and then extending the model by adding a diode, the user experiences how a simple physical system can become the foundation for data analysis and predictive modeling. By combining hands-on coding with scientific experimentation, the site invites curious learners to explore how digital tools and intelligent algorithms can emerge from the most elementary principles.
